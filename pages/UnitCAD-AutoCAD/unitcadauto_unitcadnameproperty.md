# UnitCAD Name Property

![](/images/unitcad-name-property.jpg)

This property is used to identify entities inside UnitCAD. This is then used to either locate a new entity for insertion or for actions such as move, mirror etc. For more information, refer sections on AutoCAD model Functions and Actions.
